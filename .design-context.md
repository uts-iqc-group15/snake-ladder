# Design Context

## Target Audience
- **용도**: 개발자 포트폴리오 / 데모
- **사용자**: 기술 면접관, 개발자 커뮤니티, 리크루터

## Brand Personality
- **성격**: Craft / Tabletop — 따뜻한 보드게임 질감
- **미적 방향**: 나무 질감 보드 + 종이 카드 UI + 음소거된 보석 톤 액센트
- **변경 이력**:
  - v1: Neon Arcade (2026-03-27) — AI slop 판정으로 폐기
  - v2: Craft Board (2026-03-27 확정)

## Language Support
- Korean (ko) primary
- English labels for game UI

## Accessibility
- WCAG 2.2 AA compliance
- Minimum contrast ratio: 4.5:1 (text), 3:1 (large text/UI)

## Technical Constraints
- React 19 + Vite 8 + TypeScript 6
- Tailwind CSS v4
- File naming: kebab-case
- No external UI library
- No external fonts (system fonts only)

## Current State
- Warm linen background (#f4ece0)
- Wood-toned checkerboard board (maple #e8d5b0, walnut #b8935a)
- Muted accents: crimson, navy, terracotta, forest green
- Card-style panels (no glassmorphism)
- System fonts (Georgia display, system-ui body)
- CSS 3D dice with wood tones
