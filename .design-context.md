# Design Context

## Target Audience
- **용도**: 개발자 포트폴리오 / 데모
- **사용자**: 기술 면접관, 개발자 커뮤니티, 리크루터

## Brand Personality
- **성격**: Retro (레트로) - 클래식 보드게임 감성, 빈티지 스타일
- **미적 방향**: Glassmorphism - 유리 질감 + 블러 효과의 모던 UI
- **조합 콘셉트**: Neon Arcade — 80s 아케이드 네온 팔레트 + 글래스모피즘 UI
- **선택된 방향**: A: Neon Arcade (2026-03-27 확정)

## Language Support
- Korean (ko) primary
- English labels for game UI

## Accessibility
- WCAG 2.2 AA compliance
- Minimum contrast ratio: 4.5:1 (text), 3:1 (large text/UI)

## Technical Constraints
- React 19 + Vite 8 + TypeScript 6
- **Tailwind CSS v4** (migration from CSS Modules)
- File naming: **kebab-case**
- No external UI library (vanilla components)
- CSS Modules → Tailwind v4 migration required

## Current State

- 10x10 checkerboard board (light: #f5e6ca, dark: #d4a574)
- Emoji-based snake/ladder indicators
- Basic responsive (768px breakpoint)
